## 必要なソフトウェア

### 1. Python（パイソン）のインストール

Pythonは、プログラミング言語の一つで、無料で使用できます。

**インストール手順：**

1. 以下のサイトにアクセスします：  
   [https://www.python.org/downloads/](https://www.python.org/downloads/)

2. 「Download Python」ボタンをクリックしてダウンロードします

3. ダウンロードしたファイルをダブルクリックして実行します

4. **重要：** インストール画面で「Add Python to PATH」にチェックを入れてください  
   （これにより、コマンドプロンプトからPythonを使えるようになります）

5. 「Install Now」をクリックしてインストールを完了します

### 2. 必要なPythonライブラリのインストール

Excelファイルを扱うために、追加のライブラリが必要です。

**インストール手順：**

1. コマンドプロンプトを開きます（開き方は後述）

2. 以下のコマンドを入力して、Enterキーを押します：

```
pip install openpyxl pandas
```

3. インストールが完了するまで待ちます（数分かかる場合があります）

## ファイルの準備と保存場所

### 推奨されるフォルダ構成

わかりやすく管理するために、以下のようなフォルダ構成をお勧めします：

```
C:\Excel変換プロジェクト\
├── excel_converter.py        （Pythonスクリプトファイル）
├── input\                     （変換したいExcelファイルを置くフォルダ）
│   └── data.xlsx             （変換したいExcelファイル）
└── output\                    （変換後のファイルが保存されるフォルダ）
```

### フォルダとファイルの作成手順

**1. プロジェクトフォルダの作成**

1. エクスプローラーを開きます（Windowsキー + E）

2. Cドライブを開きます

3. 何もないところで右クリック → 「新規作成」 → 「フォルダー」を選択

4. フォルダー名を「Excel変換プロジェクト」と入力します

**2. inputフォルダとoutputフォルダの作成**

1. 作成した「Excel変換プロジェクト」フォルダを開きます

2. 同様に右クリックして、「input」という名前のフォルダを作成します

3. もう一度右クリックして、「output」という名前のフォルダを作成します

**3. Pythonスクリプトファイルの保存**

1. Pythonスクリプト（excel_converter.py）を「Excel変換プロジェクト」フォルダに保存します

2. メモ帳やテキストエディタで作成した場合は、「名前を付けて保存」で保存先を「C:\Excel変換プロジェクト」にします

**4. Excelファイルの配置**

1. 変換したいExcelファイルを「input」フォルダにコピーします

2. エクスプローラーでExcelファイルを選択し、Ctrl + C（コピー）

3. 「input」フォルダを開いて、Ctrl + V（貼り付け）

## コマンドプロンプトの起動方法

コマンドプロンプトは、キーボードでコンピューターに命令を出すためのツールです。

### 方法1：検索から起動

1. Windowsキーを押します（キーボード左下にあるWindowsマークのキー）

2. 「cmd」または「コマンドプロンプト」と入力します

3. 表示された「コマンドプロンプト」をクリックします

### 方法2：ファイル名を指定して実行

1. Windowsキー + R を同時に押します

2. 「cmd」と入力してEnterキーを押します

### 方法3：エクスプローラーから直接起動（推奨）

1. エクスプローラーで「C:\Excel変換プロジェクト」フォルダを開きます

2. アドレスバー（フォルダのパスが表示されている場所）をクリックします

3. 「cmd」と入力してEnterキーを押します

→ これで、既に正しいフォルダでコマンドプロンプトが開きます！

## ディレクトリ（フォルダ）の移動方法

コマンドプロンプトで作業するフォルダを変更する方法を説明します。

### 基本的なコマンド

**1. 現在のディレクトリを確認**

```
cd
```

と入力してEnterキーを押すと、今いる場所が表示されます。

**2. ディレクトリを移動**

```
cd フォルダのパス
```

### 具体例

**例1：Excel変換プロジェクトフォルダへ移動**

```
cd C:\Excel変換プロジェクト
```

**例2：ドライブを変更する場合（DドライブやEドライブなど）**

```
D:
cd D:\Excel変換プロジェクト
```

**例3：一つ上のフォルダに戻る**

```
cd ..
```

**例4：フォルダの内容を確認**

```
dir
```

と入力すると、現在のフォルダ内のファイル一覧が表示されます。

### 便利なテクニック

- **パスのコピー＆ペースト**
  1. エクスプローラーでフォルダを開く
  2. アドレスバーのパスをクリックして選択
  3. Ctrl + C でコピー
  4. コマンドプロンプトで「cd 」と入力後、右クリックで貼り付け

- **タブキーで補完**
  - フォルダ名の最初の数文字を入力してTabキーを押すと、自動的に候補が表示されます

## Pythonスクリプトの実行方法

### 基本的な実行コマンド

コマンドプロンプトで、Pythonスクリプトがあるフォルダに移動した後、以下のコマンドを実行します：

```
python excel_converter.py
```

### 詳細な手順

**1. コマンドプロンプトを開く**
   - 前述の方法でコマンドプロンプトを起動します

**2. プロジェクトフォルダに移動**

```
cd C:\Excel変換プロジェクト
```

**3. ファイルが正しく配置されているか確認**

```
dir
```

以下のようなファイルやフォルダが表示されるはずです：
- excel_converter.py
- input
- output

**4. Pythonスクリプトを実行**

```
python excel_converter.py
```

**5. 処理の完了を待つ**
   - 画面にメッセージが表示され、処理が進行します
   - エラーメッセージが表示されなければ成功です

**6. 結果の確認**
   - エクスプローラーで「output」フォルダを開きます
   - 変換されたファイルが保存されているか確認します

### 引数を指定する場合

スクリプトによっては、変換するファイル名を指定する必要があります：

```
python excel_converter.py input\data.xlsx output\result.html
```

この場合：
- `input\data.xlsx` が変換元のExcelファイル
- `output\result.html` が変換後のファイル

## トラブルシューティング

### よくあるエラーと解決方法

#### エラー1：「'python' は、内部コマンドまたは外部コマンド...として認識されていません」

**原因：** PythonがPATHに追加されていません

**解決方法：**
1. Pythonを再インストールし、「Add Python to PATH」にチェックを入れます
2. または、以下のコマンドを試してください：
   ```
   py excel_converter.py
   ```

#### エラー2：「No such file or directory」（ファイルが見つかりません）

**原因：** ファイルが指定した場所にないか、パスが間違っています

**解決方法：**
1. `dir` コマンドでファイルの存在を確認します
2. ファイル名のスペルが正しいか確認します
3. ファイルが正しいフォルダにあるか確認します

#### エラー3：「ModuleNotFoundError: No module named 'openpyxl'」

**原因：** 必要なライブラリがインストールされていません

**解決方法：**
```
pip install openpyxl
```
または
```
pip install pandas
```

#### エラー4：「Permission denied」（アクセス拒否）

**原因：** ファイルが開かれているか、書き込み権限がありません

**解決方法：**
1. 対象のExcelファイルを閉じます
2. 管理者権限でコマンドプロンプトを開きます
   - コマンドプロンプトを右クリック → 「管理者として実行」

#### エラー5：日本語のファイル名でエラーが発生する

**原因：** 文字エンコーディングの問題

**解決方法：**
1. ファイル名を英数字のみに変更します（例：data.xlsx）
2. フォルダ名も英数字のみにします

## 補足情報

### コマンドプロンプトの基本操作

- **コマンドのキャンセル：** Ctrl + C
- **画面のクリア：** `cls` と入力してEnter
- **コマンド履歴：** ↑↓キーで過去のコマンドを呼び出せます
- **コマンドプロンプトの終了：** `exit` と入力してEnter

### ヒントとアドバイス

1. **フォルダ名やファイル名は英数字を使う**
   - 日本語を使うとエラーが発生する場合があります
   - スペースも避けた方が無難です

2. **バックアップを取る**
   - 変換前のExcelファイルは必ずバックアップを取りましょう

3. **一つずつ確認**
   - 各ステップを一つずつ確認しながら進めましょう
   - エラーが出たら、落ち着いてエラーメッセージを読みます

4. **パスの区切り文字**
   - Windowsでは `\` (バックスラッシュ) を使います
   - 例：`C:\Excel変換プロジェクト\input\data.xlsx`
